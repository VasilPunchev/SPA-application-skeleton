# SPA Application Skeleton

## Description
Basic SPA application skeleton for exam preparation.

## Project Structure
- `createSubmitHandler.js` – helper for form submissions
- `navigationUtil.js` – dynamic navigation rendering
- `renderer.js` – central render function
- `userUtils.js` – session storage and user data helpers
- `requester.js` – HTTP requester for API calls
- `userService.js` – user authentication service
- `lib.js` - use it as helper library, containing - page, html , render and nothing
  

## Features
- Modular structure
- Request handling
- User session management
- Easy rendering
- Ready for routing and views

## Usage
1. Clone the repository
2. Open the project
3. Configure the selectors and API endpoints
4. Start building views and routing

## Notes
This project is a starter skeleton and contains placeholders that need to be completed.
